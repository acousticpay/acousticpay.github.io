About AcousticPay

AcousticPay enables secure, bi-directional mobile payments using ultrasonic (inaudible) audio communication. It employs Transport Layer Security (TLS) across mobile, server, and POS devices via microphones and speakers. Encrypted data is exchanged using UTAB technology at 2,000 bps, ensuring safe and fast transactions. Each audio message contains Initiator, PublicKey, Header, and Payload for robust authentication and data transfer.
